Yummy Recipes API Documentation

Yummy recipes provides a platform for users to keep track of their awesome recipes and share with others if they so wish. To test this API live on heroku follow this link.

Authentication Â¶

This API uses OAuth v2 Bearer Token / Personal Access Token for its authentication.
